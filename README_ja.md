## 1. テスト方法

### 1.1 テスト環境

全モデルは2026年1月15日〜3月5日に [Atlas Cloud API](https://www.atlascloud.ai?ref=JPM683) 経由でアクセス。ローカル推論は未実施。

- **APIプラットフォーム**: Atlas Cloud（全モデル統一API）
- **テスト期間**: 2026年1月15日〜3月5日
- **総生成回数**: 2,847回（動画1,204、画像1,382、テキスト261）
- **総コスト**: $187.42
- **評価チーム**: 人間評価者3名 + 自動メトリクス

### 1.2 評価基準

各生成は複数のディメンションで1〜10のスケールで採点。

### 1.3 採点方法

- 各プロンプトはモデルごとに**3回実行**（異なるシード）
- スコアは3回の**中央値**
- 人間評価者が独立して採点；最終スコア = 3名の平均
- 自動メトリクス（FID、CLIPスコア）を補助検証として使用
- 拒否は0/10として採点
